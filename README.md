# 🔐 Password Analyzer

A **Password Analyzer** is a cybersecurity-focused web application that evaluates the strength of user passwords and provides recommendations to improve password security. The project helps users create strong passwords by checking various security criteria such as password length, uppercase and lowercase letters, numbers, special characters, and common password detection.

This project is developed as part of a **Cybersecurity Internship Task** to demonstrate secure coding practices and password security concepts.



## 📌 Features

- Password Strength Analysis
- Password Strength Meter
- Password Score (0–100)
- Password Requirement Validation
- Password Security Suggestions
- Common Password Detection
- Show/Hide Password
- Secure Password Generator
- Responsive User Interface
- Easy-to-understand Design

## 🎯 Project Objective

The objective of this project is to analyze user passwords based on cybersecurity best practices and educate users on creating secure passwords. The application calculates a strength score and provides suggestions to improve weak passwords.



## ⚙ How It Works

1. User enters a password.
2. The application checks:
   - Password length
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Special characters
   - Common password list
3. A password score is calculated.
4. Password strength is displayed.
5. Security suggestions are shown if needed.
6. Users can generate a strong random password.



## 📊 Password Evaluation Criteria

| Criteria | Score |
|----------|------:|
| Minimum 8 Characters | 20 |
| Uppercase Letter | 20 |
| Lowercase Letter | 20 |
| Number | 20 |
| Special Character | 20 |

Maximum Score: **100**

## ▶️ How to Execute

1. Open the project folder.
2. Double-click `index.html`.
3. Enter a password.
4. Click **Analyze Password**.
5. View the password score, strength, and suggestions.
6. Click **Generate Strong Password** to create a secure password.
